# 👥 User Management

> Managing users on the INNOVESTOR platform

---

## 📋 User List

View all registered users with:
- Name and email
- User type (Founder/Investor)
- Registration date
- Approval status

---

## 🔍 Filtering Users

### By User Type
- All Users
- Founders Only
- Investors Only

### By Status
- All
- Pending Approval
- Approved
- Rejected

---

## ✅ Approving Users

### When to Approve

Check that:
- [ ] Profile information is complete
- [ ] Name appears legitimate
- [ ] Email is valid
- [ ] LinkedIn profile exists (if provided)
- [ ] No signs of spam/fraud

### Approval Process
1. Find user in list
2. Click **"Review"**
3. Check profile details
4. Click **"Approve"** or **"Reject"**

---

## ❌ Rejecting Users

### When to Reject

- Incomplete profile information
- Suspicious/fake account
- Duplicate registration
- Policy violations

### After Rejection
- User cannot access platform features
- Can be re-reviewed later if needed

## ⚠️ Guidelines

> [!IMPORTANT]
> - Review profiles within 24-48 hours
> - Document reasons for rejection
> - Be consistent in approval criteria
> - Check for duplicate accounts
