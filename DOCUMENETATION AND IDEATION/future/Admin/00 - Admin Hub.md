## 🎯 Admin Responsibilities

As an admin, you can:
- **Review** new user registrations
- **Approve/Reject** user profiles
- **Monitor** platform activity
- **View** all ideas and transactions
- **Manage** platform integrity

## 📊 Admin Dashboard Features

### User Management
| Action | Description |
|--------|-------------|
| View Users | List all registered users |
| Filter by Type | Founders or Investors |
| Approve Profile | Enable user access |
| Reject Profile | Disable user access |

### Idea Oversight
| Action | Description |
|--------|-------------|

## ⚠️ Admin Guidelines

> [!CAUTION]
> Admin actions have platform-wide impact. Follow these guidelines:

1. **Profile Approval**
   - Verify user information is complete
   - Check for spam or fake accounts
   - Review LinkedIn profiles if provided

2. **Content Moderation**
   - Flag inappropriate ideas
   - Check for duplicate submissions
   - Ensure pitch decks are accessible

3. **Support**
   - Monitor chat request issues
   - Handle payment disputes
   - Respond to user complaints
